# ぷよぷよ

Reactで作成したぷよぷよゲームです。PC・スマートフォン両対応。

## 遊び方

### PC操作
- **← →**: 左右移動
- **↓**: 落下
- **↑**: 即落下（ハードドロップ）
- **Z**: 反時計回り回転
- **X**: 時計回り回転
- **P / Esc**: ポーズ
- **R**: リセット

### スマホ操作
画面下部のボタンで操作できます。

## ルール
- 同じ色のぷよを4つ以上繋げると消えます
- 連鎖するとスコアが大幅アップ！
- フィールドが埋まるとゲームオーバー

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```

## デプロイ

GitHub Pagesへのデプロイ:
```bash
npm run deploy
```

または、mainブランチへpushすると自動的にGitHub Actionsでデプロイされます。
